任务2:InceptionV3

CNN

卷积神经网络

pixels $\rightarrow$ edges $\rightarrow$ object parts(combination of edges) $\rightarrow$ object models

图片分为:长、宽、高(颜色)

卷积

$h(x) = \int_{-\infty}^{\infty} f(u)g(x-u)du = (fg)(x) = (gf)(x)$

$h(x) = \sum_{i = -\infty}^{\infty} x(i)h(n-i) = x(n)*h(n)$

卷积$f*g$通常比$f$和$g$更光滑 $\rightarrow$ 函数的光滑化/函数的正则化

MLP\ANN

多层感知机 Multilayer Perceptron \人工神经网络 Artificial Neural Network

输入层-隐藏层-输出层

https://www.cnblogs.com/QinHaotong/p/9571122.html

“ 用ReLU代替了Sigmoid，发现使用 ReLU 得到的SGD的收敛速度会比 sigmoid快很多，这成了AlexNet模型的优势之一 “

池化POOLING

卷积时不压缩长宽

池化就是去除杂余信息，保留关键信息
